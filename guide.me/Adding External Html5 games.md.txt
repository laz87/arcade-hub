## âœ… Legal Checklist

Before adding any game, verify:

- [ ] **License allows redistribution**
  - Look for LICENSE file
  - Check README
  - Common good licenses: MIT, Apache 2.0, GPL, CC0

- [ ] **Attribution requirements**
  - Some licenses require crediting author
  - Add credits page if needed

- [ ] **Commercial use allowed**
  - If you plan to add ads
  - Check license carefully

- [ ] **Modifications allowed**
  - Can you change the code?
  - Can you rebrand?

- Contact the author
- Don't use if uncertain
- Find alternatives
